{{ project_name }} WordPress Theme Read Me

This document describes how to install and use your WordPress theme.

Summary

  1. Plugins

    1. Activate the "Hello World" plugin

      Go to Settings » Hello World and setup plugin options

  2. Content Management

    1. Custom Post Types

      • FAQs

        Use the title of the posts as the questions and the contents of the posts as the awnsers.

      • Tours

        On preview tab, set information that will be viewed in the listing page.

        In Header Options tab you can manage the information in the header on single pages.

        In Gallery tab you can manage the images viewed as section.

        In Includes tab you can manage the includes options in each tour.

    2. Page Templates

      • Default Template

        This is the template that will be used by all pages by default.

      • Home

        The home page is divided into five sections.

      • About/Centered/Contact/Destination/FAQ

        For each section you can manage the content from the according tab.

    3. Theme Options

      Go to Theme Options and setup theme options

      • Header Settings

        Choose image that will be default image for the header if is not set.

      • Footer Settings

        Set a copyright text in the footer.

        Set the content in the left and right column of the footer.

    4. Shortcodes

      • [year]

        Use this shortcode to display the current year.

      • (*word*)

        You can use this shortcode in titles to colorize in red a word from the title.

        Example: Who (*we*) are.

        Result: Who we are.
